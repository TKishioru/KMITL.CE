คำสั่งอ่านไฟล์

```cmd
# grep -i "sshd" /var/log/authlog
```
คำสั่งค้นหาคำ "sshd"    > ดูว่าใคนบุกรุก ssh เข้ามา
ทั้งนี้ คำที่หานั้น สามารถหาได้ทั้ง SSHD SShd (ไม่สนใจพิมพ์เล็ก-ใหม่)

```cmd
# od -x /etc/issue | head -10
```
